프로그래머란?

프로그래머 : 소프트웨어 개발자는 소프트웨어 개발 작업에 종사하는 사람을 말한다.좁게는 소프트웨어 설계와 코딩
,넓게는 프로젝트 관리 업무를 수행하는 사람을 포함한다. 개별 프로그램 작업을 넘어서 응용 프로그램의 수준에서
전반적인 프로젝트에 기여하는 사람을 뜻하기도 한다.

1. QA개발자 : 다른 개발자가 쓴 코드를 체크하고 방대한 코드들에서 오류를 찾아내 고치는 등의 일을 함
2. 프론트엔드 개발자 : 우리 눈에 보이는 모든 웹을 개발함, 네이버 구글등 들어가면 보이는 모든 것
3. 백앤드 개발자 : 사용자의 눈에 보이지 않는 데이터등을 관리하고 개발함
4. Gaming play 개발자 : 말 그대로 게임 속 움직임이 보이는 모든 화면을 구연함
5. Security 보안 개발자 : 보안을 담당함
6. Data science 개발자 : 빅데이터를 담당함
